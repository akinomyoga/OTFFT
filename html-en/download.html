Download of OTFFT

OTFFT is a high-speed FFT library using Stockham algorithm and Intel AVX and Six-Step FFT algorithm and C++ template meta-programming. In addition, OTFFT is the multi-thread FFT using OpenMP. And, OTFFT has the interface for C++ and C99 and Fortran2003. The license for OTFFT is the MIT license. Please download from the following links.

  • otfft-9.5e.tar.gz (for mac/linux/cygwin)[2016/10/22]
  • otfft-9.5ev.zip (for Visual Studio 2015 nmake)[2016/10/22]

OTFFT is being developed by Japanese. The version 9.5e and 9.5ev are English version. Documents are written in English.
