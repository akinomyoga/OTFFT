Download of OTFFT

OTFFT is a high-speed FFT library using Stockham algorithm and Intel AVX/AVX-512 and Six-Step FFT algorithm and C++ template metaprogramming. In addition, OTFFT is the multi-thread FFT using OpenMP. And, OTFFT has the interface for C++ and C99 and Fortran2003. The license for OTFFT is the MIT license. Please download from the following links.

  • otfft-11.5e.tar.gz (for mac/linux)[2019/9/9]
  • otfft-11.5ev.zip (for Visual Studio 2019 nmake)[2019/9/9]
  • otfft-11.5es.zip (Solution for Visual Studio 2019)[2019/9/9]
